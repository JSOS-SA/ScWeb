# برنامج ادارة حافلات

برنامج تسجيل ومتابعة حافلات الرحلات وقياس مؤشرات الأداء

**الإدارة العامة لعمليات الساحات الخارجية **

---

## التشغيل

افتح ملف `index.html` مباشرة في متصفح حديث (Chrome أو Firefox أو Edge بإصدار حديث). لا يحتاج خادماً، لا يحتاج اتصال إنترنت بعد التنزيل، لا يحتاج تثبيت.

**ملاحظة مهمة**: المتصفحات تمنع `file://` من تحميل وحدات JavaScript (ES Modules) مباشرة. لتشغيل النظام محلياً يلزمك خادم HTTP محلي بسيط. أبسط طريقة:

```
# داخل مجلد المشروع
python -m http.server 8080
```

ثم افتح: `http://localhost:8080/`

## المبادئ المعمارية المطبقة

١. **كل ملف لا يتجاوز ٣٠٠ سطر** (التزام صارم)

٢. **فصل المسؤوليات في طبقات**:
   - طبقة الثوابت (`core/constants.js`, `core/columns.js`, `core/lists.js`): بيانات صرفة بدون منطق
   - طبقة قاعدة البيانات (`db/`): تهيئة وتخزين فقط
   - طبقة المنطق النقي (`core/shifts.js`, `core/diagnostics.js`, `core/validators.js`, `utils/`): دوال نقية بدون آثار جانبية
   - طبقة الخدمات (`services/`): تستخدم القاعدة + المنطق النقي
   - طبقة الواجهة (`ui/`): تستخدم الخدمات

٣. **منع التعشيش**: كل دالة لها مسؤولية واحدة. لا حلقات داخل حلقات داخل شروط

٤. **فصل البيانات عن التنفيذ**: ملفات `constants.js` و `columns.js` و `lists.js` تحوي بيانات فقط (`Object.freeze`). ملفات `database.js` و `seed.js` و `records.js` تحوي أوامر تنفيذ

٥. **الواردات والصادرات الواضحة**: كل ملف يُصدّر دوالاً وثوابت محددة، ويستورد فقط ما يحتاج

## الحالة الحالية للتطوير

**المرحلة الأولى مكتملة** (الأساس البنيوي):
- ✅ تهيئة قاعدة البيانات IndexedDB بـ ١٦ جدول
- ✅ تعريف الـ ٦٤ عمود لجدول السجلات
- ✅ القوائم المرجعية الأولية (١٧ نوع قائمة)
- ✅ منطق النوبات الثلاث والتشخيصات
- ✅ قواعد التحقق من الصحة
- ✅ خدمة استيراد ملف الإكسل
- ✅ الصفحة الرئيسية مع الإحصاءات الحية
- ✅ شاشة الاستيراد مع شريط تقدم

**المراحل القادمة**:
- شاشة السجل الرئيسي (Tabulator مع تحرير مباشر)
- لوحة الرحلات اللحظية
- لوحة مؤشرات الأداء
- قوالب المحاضر (Paged.js)
- محرك الصيغ HyperFormula

## كيفية البدء بعد التشغيل الأول

١. افتح `index.html`. ستظهر الصفحة الرئيسية فارغة (لا بيانات بعد)

٢. انتقل إلى **استيراد ملف إكسل** من الشريط الجانبي

٣. ارفع ملف `2026_05_05_B.xlsx` أو أي ملف بنفس البنية

٤. انتظر اكتمال الاستيراد (شريط تقدم يظهر)

٥. ارجع للصفحة الرئيسية - ستشاهد الإحصاءات معبأة
